Intro

Una de las novedades de Quarto es la posibilidad de usar extensiones. Las extensiones se escriben en el lenguaje Lua. Aquí la documentación oficial.

Vamos a aprender a usar las extensiones con un ejemplo: vamos a instalar la extensión “quarto-youtube-embed” que sirve para incluir vídeos de youtube en un documento .qmd.


Un ejemplo

Con Rmd alguna vez he puesto vídeos en mis slides o tutoriales. Usaba el paquete de R vembed. Ahora, con Quarto, también podríamos hacerlo de esa forma, solo tendríamos que insertar en nuestro documento un chunk como:

```{r}
#| eval: false
library("vembedr")
embed_url("https://www.youtube.com/watch?v=ACv9zaBa1A4")
```

Ahora, con Quarto, podemos también hacerlo usando la extensión “quarto-youtube-embed”. Aquí está el repo de la extensión. Veamos cómo:

Tarea 5.4: Instalar y usar una extensión

Concretamente instalaremos la extensión “quarto-youtube-embed”. El repo aquí. Aunque realmente el repo ha sido archivado. Veremos si se puede seguir instalando.

  • Lo primero es instalar la extensión de Quarto. Como pone en el repo, lo haremos desde la Terminal así:
quarto install extension sellorm/quarto-youtube-embed
  • Una vez instalada, podemos incrustar un video usando el siguiente “shortcode” en nuestro documento .qmd:
{{< youtube zk9zHqXm6m0 >}}

Con lo que habremos incrustado el video en nuestro documento .html final.


El shortcode << video >>

La extensión “quarto-youtube-embed” funcionaba perfectamente pero se ha quedado obsoleta porque Quarto ya incorpora de serie una extensión para incrustar vídeos. Solo hay que usar el siguiente shortcode:

{{< video https://www.youtube.com/embed/wo9vZccmqwc >}}


Aquí la documentación oficial del shortcode {{ < video >}}. Como ves, se ha mejorado la extensión original ya que ahora permite, no sólo insertar vídeos de youtube, sino también de otros servicios como Vimeo, y puedes fijar el minuto en el que empieza el vídeo y algunas cosas más.

Tarea 5.5: Instalar y usar la extensión “Lightbox”

Vamos a instalar y usar una nueva extensión, concretamente la extensión “Lightbox”. Su repo aquí

  • Lo primero es instalar la extensión desde la Terminal:
quarto install extension quarto-ext/lightbox

Esta vez, para poder usar la extensión en un documento .qmd hemos de incluir en el yaml del documento lo siguiente:

---
title: Simple Lightbox Example
filters:
   - lightbox
---

![Mi imagen con Lighbox activado](./imagenes/economia.gif){.lightbox description="El logo-nano de la facultad de Economía de la Universtat de València"
desc-position="left"}

Otro ejemplo (tweets)

Voy a instalar una nueva extensión, concretamente la extensión Quarto social embeds. Concretamente con esta extensión podremos insertar en nuestros documentos: gists de GitHub, videos de Loom, youtube y Vimeo y tweets. Realmente la instalo porque quiero tenerla disponible para insertar tweets.

En el repo se explica con más detalle lo que se puede hacer con Quarto social embeds.

Instalar y usar la extensión Quarto social embeds

Vamos a instalar y usar una nueva extensión, concretamente la extensión “Quarto social embeds”. Su repo está aquí.

  • Lo primero es instalar la extensión desde la Terminal:
quarto install extension sellorm/quarto-social-embeds


Para insertar un tweet has de:

{{< tweet user=<username> id=<id> >}}

por ejemplo:

{{< tweet user=Culturas2_tve id=1589703143562309632 >}}



Para insertar un gist de Github has de:

{{< gist sellorm 20525aff669acafb975b7495b8f2066e >}}
